## Design Philosophy

The design system is built around the concept of **"Kaze no Manga"** (Wind of Manga), emphasizing:

- **Fluidity**: Smooth, flowing interactions
- **Readability**: Clear typography and comfortable spacing
- **Accessibility**: Inclusive design for all users
- **Consistency**: Unified visual language across platforms

## Usage in Projects

- **Web (React)**: Use Tailwind preset + CSS variables
- **Mobile (Flutter)**: Reference design tokens for theme creation
- **Documentation**: This Storybook for design guidelines
